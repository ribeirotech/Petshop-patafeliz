# PataFeliz Petshop — Arquivo de Ajuda

## Sobre o Projeto

Este projeto é um sistema web front-end desenvolvido como trabalho acadêmico para a disciplina de Desenvolvimento Web. O site representa um petshop fictício chamado **PataFeliz**, com foco em apresentar produtos e serviços para animais de estimação.

## Objetivo e Metas

### Objetivo
Desenvolver um sistema web estático para um petshop, apresentando produtos categorizados e serviços disponíveis, com estrutura semântica em HTML puro, sem CSS customizado ou JavaScript (Fase 1).

### Metas
- Criar estrutura HTML semântica com `<header>`, `<main>`, `<footer>` e `<nav>`
- Apresentar 2 produtos de cada uma das 3 categorias (alimentos, acessórios, higiene), com foto, descrição e valor
- Listar todos os serviços com descrição, valor e indicação de tele-busca
- Publicar o projeto no GitHub e hospedar via GitHub Pages
